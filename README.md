## Server Side

### Usage

> python server.py [ip-address] [port]

#### or

> python server.py [ip-address]

#### or

> python server.py

#### Default values
>ip = 127.0.0.1<br/>
>port = 50500

## Client Side

### Usage

> python client.py [nickname] [ip-address] [port]

#### or

> python client.py [nickname] [ip-address]

#### or

> python client.py [nickname]

#### or

> python client.py

#### Default values
>nickname = Undefined<br/>
>ip = 127.0.0.1<br/>
>port = 50500
